# 守店猫 shoudianmao 🐱
AI桌面电商助手 | 数据解析-智能诊断-行动指引

## 项目概述
面向电商卖家的智能桌面助手，自动解析店铺经营数据，结合大模型与案例库分析问题，以萌宠状态直观展示店铺运营情况，同步输出可落地的每日运营动作。

## 核心技术关键词
LLM大模型推理 | RAG知识库检索 | 规则判定引擎
Electron桌面应用 | 浏览器数据采集 | 离线模拟演示

## 四层技术架构
数据采集层 → AI推理层 → 情绪拟人层 → 桌面展示层
1. 数据层：插件抓取电商后台经营数据，统一格式处理
2. 智能层：大模型定位异常根源，检索历史案例匹配方案
3. 交互层：根据经营状态切换宠物情绪，生成通俗运营日报
4. 展示层：桌面悬浮宠物窗口，实时可视化店铺状态

## 核心实现功能
1. 经营数据智能解读，量化指标通俗化翻译
2. 数据异常自动排查，精准定位运营问题根源
3. 历史案例参考匹配，提供成熟解决思路
4. 拟人化状态展示，萌宠情绪对应店铺好坏
5. 一键生成运营日报，输出3条当日执行任务
6. 支持离线演示，无需密钥即可完整运行

## 技术栈
运行环境：Node.js
AI能力：LLM + RAG检索 + 规则引擎
桌面客户端：Electron
数据采集：Chrome扩展插件

## 快速启动命令
# 安装项目依赖
npm install

# 运行演示程序
npm run demo
